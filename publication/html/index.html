Configuration

Launch Recipe

vllm serve stepfun-ai/Step-3.7-Flash-NVFP4 \
  --served-model-name step3p7 \
  --tensor-parallel-size 2 \
  --distributed-executor-backend ray \
  --disable-custom-all-reduce \
  --quantization modelopt \
  --kv-cache-dtype fp8 \
  --disable-cascade-attn \
  --gpu-memory-utilization 0.70 \
  --max-model-len 8192 \
  --reasoning-parser step3p5 \
  --tool-call-parser step3p5 \
  --enable-auto-tool-choice \
  --enforce-eager

Known Limitations

Caveats

  • nsys cannot trace Ray worker GPU kernels across nodes (distributed TP limitation)
  • --enforce-eager required (Triton JIT + CUDA graphs cause RPC timeout on first run)
  • max_model_len=8192 (conservative; model supports 256K)
  • Upstream recommends TP=4+EP; TP=2 is our validated target
  • Vision test uses synthetic base64 PNG
  • PYNCCL all-reduce (SM121 NCCL_SYMM_MEM unavailable)
